Matplotlib

Matplotlib 是一个进行数据展示的 2D 绘图库。

Matplotlib 3.0 只支持 Python 3；Matplotlib 2.2.x 支持 Python 2，不过 bug 修复和发布到 2020 年 1 月 1 日将停止。
