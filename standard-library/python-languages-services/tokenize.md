tokenize
========
[TOC]
> token: n. 象征；标志； adj. 作为标志的；
> -ize: suff. 使成...状态；使...化；
> tokenize：标识化；标记化；

`tokenize` 提供了“**对 Python 代码使用的**”词汇扫描器，是用 Python 实现的。扫描器可以给 Python 代码打上标记后返回，你可以看到每一个词或者字符是什么类型的。扫描器甚至将注释也单独标记，这样某些需要对代码进行特定风格展示的地方就很方便了。
